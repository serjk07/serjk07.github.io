Privacy Policy

Last updated: January 2026

1. Information We Collect

The application stores OAuth access tokens locally on the user's device. No personal data is transmitted to any external server except TikTok’s official API.

2. How We Use Data

Access tokens are used exclusively for authorized video publishing via TikTok’s official Direct Post API.

3. Data Storage

All data is stored locally on the user’s computer. We do not operate any cloud storage.

4. Third-Party Services

The application interacts only with TikTok's official API endpoints.
